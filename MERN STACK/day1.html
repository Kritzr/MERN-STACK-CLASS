MERN Stack Overview

What is MERN Stack?

The MERN stack is a combination of technologies used to build modern web applications:

  • Client: React
  • Server: Node.js and Express.js
  • Database: MongoDB

The UI and UX are important aspects of web development that focus on User Interface and User Experience.

Frontend development can be done using frameworks like Angular, React, or Flutter. Frontend focuses on displaying the interface, and it does not store data.

The frontend and database do not interact directly. The server coordinates the communication between them.

HTML Basics

HTML is used to structure content on the web. It consists of tags that tell the browser how to render content.

containers and standalone tags

Some tags in HTML are containers (they have opening and closing tags), such as:

  • <div> - Container for grouping content
  • <p> - Paragraphs of text
  • <table> - Tables for data

standalone tags do not have closing tags, such as:

  • <br> - Line break
  • <hr> - Horizontal rule

What are Input Tags?

Input tags in HTML allow users to provide data. Examples include:

  • <input> - Standard input fields
  • <textarea> - Multi-line input fields for text
