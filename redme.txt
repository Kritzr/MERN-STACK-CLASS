What is a Callback Function?
A callback function is a function passed as an argument to another function, and itâ€™s called (executed) later inside that outer function.

It allows you to control the order of execution, especially in asynchronous code (like fetching data, event listeners, or timers).

 Why use callbacks?
Because JavaScript is non-blocking and runs code asynchronously. So we need a way to say, â€œHey, run this after you're done!â€

 Basic Example
js
Copy
Edit
function greet(name, callback) {
  console.log("Hi, " + name);
  callback();  // calling the callback function
}

function sayBye() {
  console.log("Bye!");
}

greet("Krithika", sayBye);
Output:


Edit
Hi, Krithika
Bye!
Here, sayBye is passed as a callback and executed after greeting.

With Asynchronous Code (like setTimeout)
js
Copy
Edit
setTimeout(function () {
  console.log("This runs after 2 seconds");
}, 2000);
Here, the anonymous function is a callback that runs after 2 seconds.
